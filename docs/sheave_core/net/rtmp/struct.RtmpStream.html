Struct RtmpStream 

Source 
pub struct RtmpStream { /* private fields */ }
Expand description

A stream for RTMP that wrapped Tokio’s TcpStream.

If you constructs this struct from some address, use RtmpStream::connect("aaa.bbb.ccc.ddd:1935"). Or if you do it from already created std’s TcpStream, use RtmpStream::from_std(std_stream)
