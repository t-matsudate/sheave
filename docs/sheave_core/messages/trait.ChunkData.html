Expand description

Uniforms chunk data by a channel numbers and a message type.

When reading from streams or writing into streams, we are required to imprint a chunk ID and a message type ID into their headers. This makes you to reduce its cost. For example, read_chunk and write_chunk use this for reading/writing chunks correctly.
