The command to tell publishing information.

Following format is required:

FieldAMF TypeValue
NullNothing but an AMFโs type marker is in.
Publishing NameStringA name for publishing a data to the server.
Publishing TypeStringSee Publishing Type.

ยงPublishing Type

The publish command requires you to specify one of โPublishing Typeโ in its request. Publishing Type means:

Pubishing TypeDescription
"live"Only streaming.
Its data will never be stored.
"record"Its data will be stored.
If publishing name duplicated, it is rewritten as a new file.
"append"Same as "record" excepts is appended its data if publishing name duplicated.
