Enum PlayMode

#[repr(i64)]
pub enum PlayMode {
    Other = -9_223_372_036_854_775_808,
    Both = -2,
    Live = -1,
    Record = 0,
}
Expand description

Representation of the way to subscribe its stream.

Variants correspond to respectively following numbers:

PatternNumber
Otherbelow than -2
Both-2
Live-1
Recordedabove 0

Note this constins signed numbers above -2.
