pub struct ByteBuffer { /* fields omitted */ }

The byte buffer to encode/decode the Rust's data/the bytes

This consists of following data:

FieldType
offsetusize
lenusize
bytesVec<u8>

This is implemented to premise that will dispose after receiving/sending the chunk. Therefore this is considered just to append the bytes, not to overwrite its bytes.
